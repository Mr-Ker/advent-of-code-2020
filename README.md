# Advent of code - 2020

```
usage: adventofcode.py [-h] [-d DAY] [-i INPUT] [--extra-args [EXTRA_ARGS [EXTRA_ARGS ...]]]

Advent of code 2020

optional arguments:
  -h, --help            show this help message and exit
  -d DAY, --day DAY     Day of advent of code to run
  -i INPUT, --input INPUT
                        Input file for the day's exercise
  --extra-args [EXTRA_ARGS [EXTRA_ARGS ...]]
                        Extra arguments specific for the day's exercise.
                        Use [--extra-args help] to list available extra arguments for the day.
```
